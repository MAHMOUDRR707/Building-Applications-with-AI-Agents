# Building Applications with AI Agents

A hands-on learning repository based on the book **Building Applications with AI Agents**.

This repository documents my journey through the book chapter by chapter, including:

- Chapter summaries
- Notes and key takeaways
- Practical exercises
- Small projects and implementations
- Additional experiments and explorations

The goal is not only to understand the concepts presented in the book, but also to apply them through real implementations and code examples.

---

## Book

This repository accompanies the O’Reilly Media book
[**Building Applications with AI Agents: Designing and Implementing Multi-Agent Systems**](https://www.oreilly.com/library/view/building-applications-with/9781098176495/).
All scenarios, architectural patterns, and observability examples reflect the lessons and code samples from the book. Feel free to follow along chapter by chapter or dive straight into the code!

## Learning Objectives

Through this repository, I aim to:

- Understand the fundamentals of AI agents
- Explore agent architectures and design patterns
- Learn orchestration and workflow management
- Build agent-powered applications
- Experiment with memory, planning, and tool usage
- Practice multi-agent collaboration patterns
- Gain hands-on experience with modern agent frameworks
